# grill-me

Relentless design interviewer that walks every branch of a decision tree until reaching shared understanding.

## Usage

Triggers when the user mentions "grill me", wants to stress-test a plan, or asks for adversarial design review.

## What it does

- Interviews the user about every aspect of a plan or design
- Resolves decision branches one by one, handling dependencies in order
- Provides a recommended answer for each question
- Explores the codebase instead of asking when a question can be answered that way
